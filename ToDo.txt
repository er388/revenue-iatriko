Τα reports, comparison, forecasting, heatmaps, cloud μπορεί να τα φαίρεσες για απλοποίηση, όμως είναι αναγκαία. Υπάρχει άλλο implementation; Αν όχι, πρέπει να ξαναμπούν.
Τα πάντα είναι μεγάλα. Δεν χρειάζεται να είναι τόσο τεράστια τα κουμπιά. Να γίνει λίγο πιο συμμαζεμμένο. Ένα 20% ίσως πιο μικρά στοιχεία;
Το theme button να μπει μέσα στις ρυθμίσεις.
Τα μπλε πεδία με τα ποσά να μαζευτούν όλα μαζί κάτω από την καταχώριση, όχι άλλα επάνω και άλλα κάτω.
Όπου υπάρχουν ποσά κρατήσεων θα πρέπει να υπάρχει και αντίστοιχο πεδίο με το ποσοστό επί του ποσού στο οποίο γίνεται η κράτηση και να είναι αλληλοσυμπληρούμενο. Ό,τι βάλω στο ένα να εμφανίζεται αυτόματα το άλλο.
Στην καταχώριση να παραμένουν οι τελευταίες επιλογές στο Διαγνωστικό, στην Ασφάλεια και στον Τύπο Είσπραξης για γρήγορη καταχώριση. Τα + στη βασική καρτέλα στην καταχώριση δε χρειάζονται, μιας και η διαχείριση γίνεται στις ρυθμίσεις.
Στον Τύπο Είσπραξης να διορθωθεί το Μετρητά (…) σε Μετρητά σκέτο, εννοούνται οι κάρτες.
Όπου χρειάζεται να εισαχθεί ημερομηνία, η μορφή μμ/εε να γίνεται αποδεκτή και να διορθώνεται σε μμ/εεεε αυτόματα. Η αυτόματη εισαγωγή του / να παραμείνει.
Όταν γίνεται καταχώριση (είτε ΕΟΠΥΥ είτε άλλη ασφάλεια) να υπάρχει και ένα πεδίο που να συμπληρώνεται αυτόματα και να αναγράφει το τελικό ποσό για γρήγορο έλεγχο από τον χρήστη.
Οι σημειώσεις να είναι πιο μικρό το πεδίο, δεν ξέρω καν αν χρειαστούν. Μήπως να βάλουμε ένα check box και αν χρειαστεί να το πατήσω και να εμφανιστεί το πεδίο;
Στις εγγραφές οι κρατήσεις να είναι σε δεύτερη γραμμή, ώστε να μπορέσουν να χωρέσουν τα ποσοστά.
Οι πρόσφατες εγγραφές να είναι dropdown list και default να είναι μαζεμένες. Να γίνει σωστά η στοίχιση.
Export pdf όπου μπορεί να γίνει: εγγραφές, charts, κλπ. Οπουδήποτε.
Στο κουμπί + Νέα εγγραφή επίσης να γίνουν οι παραπάνω αλλαγές.
Στα φίλτρα να προστεθεί φίλτρο από - έως ποσό και από - έως ποσοστό.
Στις ρυθμίσεις η στρατηγική και η χρήση δε χρειάζονται.
Το autosave να γίνεται κάθε 5 αλλαγές της βάσης δεδομένων, όχι με χρόνο.
Η επικίνδυνη ζώνη να μεταφερθεί στο τέλος της σελίδας.
Οι ασφάλειες και τα διαγνωστικά να μπορούν να μπουν με όποια σειρά θέλω εγώ, όχι μόνο όπως καταχωρούνται.
